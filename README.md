# StyleShop - Tienda de Ropa Premium

Tienda online profesional desarrollada con HTML5, CSS3 y JavaScript vanilla. Una aplicación moderna de comercio electrónico que permite a los usuarios explorar, filtrar y comprar productos de ropa de alta calidad con una experiencia de usuario excepcional.

## Descripción del Proyecto

StyleShop es una tienda de ropa virtual que ofrece una interfaz intuitiva y moderna para la compra de prendas. La aplicación incluye un sistema completo de carrito de compras, filtrado de productos por categorías y un proceso de compra simple y directo.

## Características Principales

### Interfaz de Usuario
- Navbar sticky con navegación y carrito integrado
- Hero section con gradiente atractivo
- Diseño responsive para todos los dispositivos
- Footer con información de contacto
- Animaciones suaves y transiciones

### Gestión de Productos
- Catálogo de 10 productos con imágenes
- Sistema de categorías (Remeras, Buzos, Accesorios, Especiales)
- Filtrado dinámico por categoría
- Información detallada de cada producto (nombre, descripción, precio)
- Badges para destacar productos nuevos

### Funcionalidades del Carrito
- Agregar productos al carrito con un clic
- Modificar cantidades (aumentar/disminuir)
- Edición directa de cantidad mediante input
- Eliminar productos individuales
- Cálculo automático de subtotales y total
- Contador de items en tiempo real
- Modal elegante para visualizar el carrito
- Mensaje de confirmación al completar la compra
- Vaciado automático del carrito después de comprar

### Sistema de Precios
- Precios formateados en pesos argentinos
- Cálculo automático de subtotales
- Total actualizado en tiempo real
- Visualización clara del desglose de costos

## Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Estilos avanzados con gradientes, sombras y animaciones
- **JavaScript (Vanilla)** - Lógica de la aplicación sin dependencias externas
- **Font Awesome 6.4** - Iconografía profesional
- **Variables CSS** - Paleta de colores consistente

## Cómo Usar

1. Abre `index.html` en tu navegador web
2. Explora los productos disponibles
3. Usa los filtros para buscar por categoría
4. Haz clic en "Agregar al Carrito" para añadir productos
5. Visualiza tu carrito haciendo clic en el icono del carrito en el navbar
6. Ajusta las cantidades según sea necesario
7. Haz clic en "Proceder al Pago" para completar la compra
