# Carrito de Compras

Aplicación web de carrito de compras desarrollada con HTML, CSS y JavaScript. Permite agregar productos, modificar cantidades, eliminar items y calcular el total automáticamente.

## Características

- Agregar productos al carrito
- Aumentar o disminuir cantidades
- Eliminar productos
- Cálculo automático del total
- Interfaz responsiva y amigable
